常用方法

  • Thread 的常用方法
  • start () 执行当前线程
  • run () 重新 Thread 的 run 方法，在此方法中写需要线程执行的内容
  • currentThread ()，静态方法，返回当前线程
  • getName () 获取线程名称
  • setName () 设置线程名称
  • yield 释放当前 CPU 执行权
  • join () 线程进入阻塞状态，需要等其他线程执行完才继续执行
  • sleep () 线程休眠，单位毫秒
  • isAlive () 判断线程是否存活

    注意

  1. 不能在主线程调用 run 方法启动，必须调用 start 方法，否则无法实现多线程
  2. 不能在已 start 的线程再次 start
