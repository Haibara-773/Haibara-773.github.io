什么是 Servlet

  1. Servlet 是 JavaEE 规范之一。规范就是接口

  2. Servlet 就是 Java Web 三大组件之一

    三大组件：

    • Servlet 程序
    • Fulter 过滤器
    • Listener 监听器

  3. Servlet 是运行在服务器上的一个 Java 小程序，可以接收客户端发送过来的请求，并且响应数据
